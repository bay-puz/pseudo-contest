ヤジリンのルール

  1. 線を引いて、盤面全体で輪っかを１つ作りましょう。
  2. 線はタテヨコに引いてマスの中央を通り、交差も枝分かれもしません。
  3. 線が通らず、ヒント（矢印付きの数字と「？」）もないマスは黒マスになります。
  4. 数字は、そのマスから矢印が向いている方向にある黒マスの数を表します。
  5. 黒マスはタテヨコに連続しません。

ヤジリンのアンサーキー

アンサーキーは、解答盤面で輪っかの内側にある黒マスの数です。
